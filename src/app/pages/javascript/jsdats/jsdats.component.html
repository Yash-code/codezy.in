JavaScript Dates

JavaScript Date Output


By default, JavaScript will use the browser's time zone and display a date as a full text string: Fri Aug 07 2020 17:31:52 GMT+0530 (India Standard Time)


var d = new Date();

Creating Date Objects


Date objects are created with the new Date() constructor. There are 4 ways to create a new date object:

new Date()
new Date(year, month, day, hours, minutes, seconds, milliseconds)
new Date(milliseconds)
new Date(date string)


JavaScript Stores Dates as Milliseconds


SJavaScript stores dates as number of milliseconds since January 01, 1970, 00:00:00 UTC (Universal Time Coordinated).


Now the time is: 1596801712987 milliseconds past January 01, 1970


new Date(milliseconds)


new Date(milliseconds) creates a new date object as zero time plus milliseconds:

var d = new Date(0);
